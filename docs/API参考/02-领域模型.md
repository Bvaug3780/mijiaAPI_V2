## Credential（凭据）

用户认证凭据，是整个系统的核心对象。

### 定义

```python
class Credential(BaseModel):
    """用户凭据
    
    包含用户认证所需的所有信息。
    """
    user_id: str          # 用户ID
    service_token: str    # 服务令牌
    ssecurity: str        # 安全密钥
    expires_at: int       # 过期时间戳（秒）
```

### 属性说明

#### user_id
- **类型**：`str`
- **说明**：用户的唯一标识符
- **用途**：
  - 缓存命名空间隔离
  - 日志追踪
  - 多用户场景的用户区分

#### service_token
- **类型**：`str`
- **说明**：服务访问令牌
- **用途**：API请求的认证凭证
- **注意**：敏感信息，需要安全存储

#### ssecurity
- **类型**：`str`
- **说明**：安全密钥
- **用途**：
  - 请求数据加密
  - 签名生成
- **注意**：敏感信息，需要安全存储

#### expires_at
- **类型**：`int`
- **说明**：凭据过期时间戳（Unix时间戳，秒）
- **用途**：判断凭据是否过期

### 方法

#### is_expired()

检查凭据是否已过期。

```python
def is_expired(self) -> bool:
    """检查凭据是否已过期
    
    Returns:
        bool: 已过期返回True，否则返回False
    """
    import time
    return time.time() >= self.expires_at
```

**使用示例**：
```python
if credential.is_expired():
    print("凭据已过期，需要刷新")
    credential = auth_service.refresh_credential(credential)
else:
    print(f"凭据有效，剩余 {credential.expires_at - time.time()} 秒")
```

### 序列化

```python
# 转换为字典
credential_dict = credential.model_dump()

# 转换为JSON
credential_json = credential.model_dump_json()

# 从字典创建
credential = Credential.model_validate(credential_dict)

# 从JSON创建
credential = Credential.model_validate_json(credential_json)
```

### 存储建议

```python
# 不推荐：明文存储
with open("credential.json", "w") as f:
    f.write(credential.model_dump_json())

# 推荐：使用凭据存储服务
from mijiaAPI_V2.infrastructure.credential_store import FileCredentialStore

store = FileCredentialStore()
store.save(credential)  # 自动处理路径和格式
```

## Home（家庭）

表示一个米家智能家居的家庭。

### 定义

```python
class Home(BaseModel):
    """家庭实体"""
    id: str               # 家庭ID
    name: str             # 家庭名称
    uid: str              # 所有者用户ID
    rooms: List[Dict]     # 房间列表
```

### 属性说明

#### id
- **类型**：`str`
- **说明**：家庭的唯一标识符
- **用途**：获取家庭下的设备、智能等

#### name
- **类型**：`str`
- **说明**：家庭名称（用户自定义）
- **示例**：`"我的家"`, `"公司"`

#### uid
- **类型**：`str`
- **说明**：家庭所有者的用户ID
- **用途**：权限判断

#### rooms
- **类型**：`List[Dict]`
- **说明**：房间列表
- **结构**：
```python
{
    "id": "room_id",
    "name": "客厅",
    "icon": "icon_url"
}
```

### 使用示例

```python
# 获取家庭列表
homes = api.get_homes()

# 遍历家庭
for home in homes:
    print(f"家庭: {home.name}")
    print(f"  ID: {home.id}")
    print(f"  所有者: {home.uid}")
    print(f"  房间数: {len(home.rooms)}")
    
    # 遍历房间
    for room in home.rooms:
        print(f"    - {room['name']}")
```

## Device（设备）

表示一个智能设备。

### 定义

```python
class Device(BaseModel):
    """设备实体"""
    did: str                    # 设备ID
    name: str                   # 设备名称
    model: str                  # 设备型号
    home_id: str                # 所属家庭ID
    room_id: Optional[str]      # 所属房间ID
    status: int                 # 在线状态（1=在线，0=离线）
    parent_id: Optional[str]    # 父设备ID（网关设备）
    parent_model: Optional[str] # 父设备型号
    show_mode: Optional[int]    # 显示模式
    extra: Optional[Dict]       # 额外信息
```

### 属性说明

#### did
- **类型**：`str`
- **说明**：设备的唯一标识符
- **用途**：控制设备、获取设备状态

#### name
- **类型**：`str`
- **说明**：设备名称（用户自定义）
- **示例**：`"客厅灯"`, `"卧室空调"`

#### model
- **类型**：`str`
- **说明**：设备型号
- **示例**：`"yeelink.light.lamp4"`, `"xiaomi.aircondition.v1"`
- **用途**：
  - 获取设备规格
  - 判断设备类型
  - 确定支持的功能

#### home_id
- **类型**：`str`
- **说明**：设备所属的家庭ID

#### room_id
- **类型**：`Optional[str]`
- **说明**：设备所属的房间ID
- **注意**：可能为None（未分配房间）

#### status
- **类型**：`int`
- **说明**：设备在线状态
- **取值**：
  - `1`: 在线
  - `0`: 离线

#### parent_id / parent_model
- **类型**：`Optional[str]`
- **说明**：父设备信息（用于子设备）
- **场景**：网关下的子设备（如温湿度传感器）

### 使用示例

```python
# 获取设备列表
devices = api.get_devices(home_id="123456")

# 筛选在线设备
online_devices = [d for d in devices if d.status == 1]
print(f"在线设备: {len(online_devices)}/{len(devices)}")

# 按型号分组
from collections import defaultdict
by_model = defaultdict(list)
for device in devices:
    by_model[device.model].append(device)

for model, devices in by_model.items():
    print(f"{model}: {len(devices)}台")

# 查找特定设备
light = next((d for d in devices if "light" in d.model), None)
if light:
    print(f"找到灯: {light.name}")
```

## DeviceSpec（设备规格）

设备的规格信息，定义设备支持的服务、属性和操作。

### 定义

```python
class DeviceSpec(BaseModel):
    """设备规格"""
    model: str                    # 设备型号
    type: str                     # 设备类型
    services: List[ServiceSpec]   # 服务列表
```

### ServiceSpec（服务规格）

```python
class ServiceSpec(BaseModel):
    """服务规格"""
    siid: int                     # 服务ID
    type: str                     # 服务类型
    description: str              # 服务描述
    properties: List[PropertySpec] # 属性列表
    actions: List[ActionSpec]     # 操作列表
```

### PropertySpec（属性规格）

```python
class PropertySpec(BaseModel):
    """属性规格"""
    piid: int                     # 属性ID
    type: str                     # 属性类型
    description: str              # 属性描述
    format: str                   # 数据格式（bool, int, float, string）
    access: List[str]             # 访问权限（read, write, notify）
    value_range: Optional[List]   # 值范围
    value_list: Optional[List]    # 值列表（枚举）
```

### 使用示例

```python
from mijiaAPI_V2.repositories.device_spec_repository import DeviceSpecRepositoryImpl

# 获取设备规格
spec_repo = DeviceSpecRepositoryImpl(http_client, cache_manager)
spec = spec_repo.get_spec(model="yeelink.light.lamp4")

# 查看服务
for service in spec.services:
    print(f"服务: {service.description} (siid={service.siid})")
    
    # 查看属性
    for prop in service.properties:
        access = ", ".join(prop.access)
        print(f"  属性: {prop.description} (piid={prop.piid})")
        print(f"    类型: {prop.format}, 权限: {access}")
        
        if prop.value_range:
            print(f"    范围: {prop.value_range}")
        if prop.value_list:
            print(f"    可选值: {prop.value_list}")
```

## 数据验证

所有模型都使用 Pydantic 进行自动验证。

### 验证示例

```python
from pydantic import ValidationError

# 正确的数据
try:
    credential = Credential(
        user_id="123456",
        service_token="token_xxx",
        ssecurity="secret_xxx",
        expires_at=1234567890
    )
    print("验证通过")
except ValidationError as e:
    print(f"验证失败: {e}")

# 错误的数据（缺少必需字段）
try:
    credential = Credential(
        user_id="123456",
        # 缺少 service_token
        ssecurity="secret_xxx",
        expires_at=1234567890
    )
except ValidationError as e:
    print(f"验证失败: {e}")
    # 输出: Field required [type=missing, input_value=...]

# 错误的数据类型
try:
    credential = Credential(
        user_id="123456",
        service_token="token_xxx",
        ssecurity="secret_xxx",
        expires_at="not_a_number"  # 应该是int
    )
except ValidationError as e:
    print(f"验证失败: {e}")
```

## 模型转换

### 字典转换

```python
# 模型 → 字典
device_dict = device.model_dump()

# 字典 → 模型
device = Device.model_validate(device_dict)

# 排除某些字段
device_dict = device.model_dump(exclude={"extra"})

# 只包含某些字段
device_dict = device.model_dump(include={"did", "name", "model"})
```

### JSON转换

```python
# 模型 → JSON字符串
device_json = device.model_dump_json()

# JSON字符串 → 模型
device = Device.model_validate_json(device_json)

# 美化输出
device_json = device.model_dump_json(indent=2)
```

### 批量转换

```python
# 模型列表 → 字典列表
devices_dict = [d.model_dump() for d in devices]

# 字典列表 → 模型列表
devices = [Device.model_validate(d) for d in devices_dict]
```

## 模型扩展

### 自定义验证

```python
from pydantic import field_validator

class CustomDevice(Device):
    """自定义设备模型"""
    
    @field_validator('status')
    @classmethod
    def validate_status(cls, v):
        """验证状态值"""
        if v not in [0, 1]:
            raise ValueError("状态必须是0或1")
        return v
    
    @field_validator('name')
    @classmethod
    def validate_name(cls, v):
        """验证名称"""
        if not v or len(v) > 50:
            raise ValueError("名称长度必须在1-50之间")
        return v
```

### 计算属性

```python
class EnhancedDevice(Device):
    """增强的设备模型"""
    
    @property
    def is_online(self) -> bool:
        """设备是否在线"""
        return self.status == 1
    
    @property
    def brand(self) -> str:
        """设备品牌（从型号推断）"""
        return self.model.split('.')[0]
    
    @property
    def device_type(self) -> str:
        """设备类型（从型号推断）"""
        parts = self.model.split('.')
        return parts[1] if len(parts) > 1 else "unknown"
```

## 最佳实践

### 1. 使用类型提示

```python
from typing import List

def process_devices(devices: List[Device]) -> List[str]:
    """处理设备列表"""
    return [d.name for d in devices if d.is_online]
```

### 2. 验证外部数据

```python
# 从API获取的数据
api_response = {"did": "123", "name": "灯", "model": "xxx", ...}

# 使用模型验证
try:
    device = Device.model_validate(api_response)
    # 数据有效，可以安全使用
except ValidationError as e:
    # 数据无效，记录错误
    logger.error(f"设备数据无效: {e}")
```

### 3. 不可变性

```python
# Pydantic模型默认是可变的
device.name = "新名称"  # 可以修改

# 如果需要不可变模型
from pydantic import ConfigDict

class ImmutableDevice(Device):
    model_config = ConfigDict(frozen=True)

# 现在无法修改
device = ImmutableDevice(...)
device.name = "新名称"  # 抛出异常
```

### 4. 序列化配置

```python
# 自定义序列化
device_dict = device.model_dump(
    exclude_none=True,      # 排除None值
    exclude_unset=True,     # 排除未设置的字段
    by_alias=True,          # 使用别名
)
```
