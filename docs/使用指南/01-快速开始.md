# 快速开始

## 安装

### 使用 uv（推荐）

```bash
# 克隆仓库
git clone https://github.com/your-org/mijia-api.git
cd mijia-api

# 安装依赖
uv sync

# 安装开发依赖
uv sync --extra dev

# 安装可选的Redis支持
uv sync --extra redis
```

### 使用 pip

```bash
pip install -r requirements.txt
```

## 第一个程序

### 1. 获取凭据

首次使用需要登录获取凭据：

```python
from mijiaAPI_V2 import create_auth_service

# 创建认证服务
auth_service = create_auth_service()

# 方式一：二维码登录（推荐）
credential = auth_service.login_by_qrcode()
# 会显示二维码，使用米家APP扫码登录

# 方式二：账号密码登录
# credential = auth_service.login_by_password("your_username", "your_password")

# 保存凭据到文件
auth_service.save_credential(credential)
print(f"凭据已保存，用户ID: {credential.user_id}")
```

**为什么需要保存凭据？**
- 避免每次都要登录
- 凭据有效期通常为30天
- 可以在多个脚本中复用

### 2. 创建API客户端

```python
from mijiaAPI_V2 import create_api_client_from_file

# 从文件加载凭据并创建客户端
api = create_api_client_from_file()

# 或者手动创建
from mijiaAPI_V2 import create_api_client, create_auth_service

auth_service = create_auth_service()
credential = auth_service.load_credential()
api = create_api_client(credential)
```

### 3. 获取家庭列表

```python
# 获取所有家庭
homes = api.get_homes()

for home in homes:
    print(f"家庭: {home.name} (ID: {home.id})")
    print(f"  所有者: {home.uid}")
    print(f"  房间数: {len(home.rooms)}")
```

**输出示例**：
```
家庭: 我的家 (ID: 123456)
  所有者: 987654321
  房间数: 3
```

### 4. 获取设备列表

```python
# 使用第一个家庭的ID
home_id = homes[0].id

# 获取该家庭下的所有设备
devices = api.get_devices(home_id=home_id)

for device in devices:
    status = "在线" if device.status == 1 else "离线"
    print(f"{device.name} ({device.model}) - {status}")
```

**输出示例**：
```
客厅灯 (yeelink.light.lamp4) - 在线
卧室空调 (xiaomi.aircondition.v1) - 在线
厨房插座 (chuangmi.plug.v3) - 离线
```

### 5. 控制设备

```python
# 获取第一个设备
device = devices[0]

# 控制设备属性
# siid: 服务ID, piid: 属性ID, value: 属性值
success = api.control_device(
    device_id=device.did,
    siid=2,      # 灯光服务
    piid=1,      # 开关属性
    value=True   # 打开
)

if success:
    print(f"成功控制设备: {device.name}")
else:
    print(f"控制设备失败: {device.name}")
```

**如何知道 siid 和 piid？**
- 查看设备规格文档
- 使用米家开发者平台
- 参考 [领域模型参考](../API参考/02-领域模型.md) 中的 DeviceSpec 部分

## 完整示例

```python
from mijiaAPI_V2 import create_api_client_from_file

def main():
    # 1. 创建API客户端
    api = create_api_client_from_file()
    
    # 2. 获取家庭列表
    homes = api.get_homes()
    if not homes:
        print("没有找到家庭")
        return
    
    print(f"找到 {len(homes)} 个家庭")
    home = homes[0]
    print(f"使用家庭: {home.name}")
    
    # 3. 获取设备列表
    devices = api.get_devices(home_id=home.id)
    print(f"找到 {len(devices)} 个设备")
    
    # 4. 显示设备信息
    for device in devices:
        status = "在线" if device.status == 1 else "离线"
        print(f"  - {device.name} ({device.model}): {status}")
    
    # 5. 控制第一个在线设备
    online_devices = [d for d in devices if d.status == 1]
    if online_devices:
        device = online_devices[0]
        print(f"\n尝试控制设备: {device.name}")
        
        # 示例：打开设备
        success = api.control_device(
            device_id=device.did,
            siid=2,
            piid=1,
            value=True
        )
        
        if success:
            print("控制成功！")
        else:
            print("控制失败")

if __name__ == "__main__":
    main()
```

## 常见问题

### Q1: 凭据过期怎么办？

```python
from mijiaAPI_V2 import create_auth_service
from mijiaAPI_V2.domain.exceptions import TokenExpiredError

auth_service = create_auth_service()
credential = auth_service.load_credential()

# 检查是否过期
if credential.is_expired():
    print("凭据已过期，正在刷新...")
    credential = auth_service.refresh_credential(credential)
    auth_service.save_credential(credential)
    print("凭据刷新成功")
```

### Q2: 如何处理错误？

```python
from mijiaAPI_V2.domain.exceptions import (
    DeviceNotFoundError,
    DeviceOfflineError,
    TokenExpiredError
)

try:
    api.control_device(device_id="xxx", siid=2, piid=1, value=True)
except TokenExpiredError:
    print("凭据已过期，请刷新凭据")
except DeviceNotFoundError:
    print("设备不存在")
except DeviceOfflineError:
    print("设备离线")
except Exception as e:
    print(f"未知错误: {e}")
```

### Q3: 如何启用调试日志？

```python
import logging

# 设置日志级别
logging.basicConfig(level=logging.DEBUG)

# 或者只启用mijiaAPI的日志
logger = logging.getLogger("mijiaAPI_V2")
logger.setLevel(logging.DEBUG)
```

### Q4: 如何使用自定义配置？

```python
from pathlib import Path
from mijiaAPI_V2 import create_api_client

# 使用自定义配置文件
api = create_api_client(
    credential=credential,
    config_path=Path("my_config.toml")
)
```
