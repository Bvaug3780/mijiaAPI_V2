# 设备规格翻译功能

本模块提供设备属性和操作的中文翻译功能，帮助用户理解设备功能。

## 功能特性

- 内置常见属性和操作的中文翻译
- 支持从外部 JSON 文件加载自定义翻译
- 支持通过代码传入自定义翻译字典
- 支持动态添加翻译
- 支持翻译优先级：自定义字典 > 外部文件 > 内置翻译

## 使用方法

### 1. 使用默认翻译（最简单）

```python
from mijiaAPI_V2.repositories.property_translations import (
    get_property_translation,
    get_action_translation,
)

# 获取属性翻译
cn_name = get_property_translation("Switch Status")  # 返回: "开关状态"

# 获取操作翻译
cn_name = get_action_translation("Toggle")  # 返回: "切换开关"
```

### 2. 使用自定义翻译字典

```python
from mijiaAPI_V2.repositories.property_translations import TranslationManager

# 创建自定义翻译
custom_translations = {
    "properties": {
        "my-property": "我的属性",
        "another-property": "另一个属性",
    },
    "actions": {
        "my-action": "我的操作",
    }
}

# 创建翻译管理器
translator = TranslationManager(custom_translations=custom_translations)

# 使用翻译
cn_name = translator.get_property_translation("my-property")  # 返回: "我的属性"
cn_name = translator.get_property_translation("Switch Status")  # 返回: "开关状态" (内置翻译)
```

### 3. 从外部文件加载翻译

首先创建翻译文件 `my_translations.json`：

```json
{
  "properties": {
    "my-property": "我的属性",
    "another-property": "另一个属性"
  },
  "actions": {
    "my-action": "我的操作"
  }
}
```

然后加载：

```python
from pathlib import Path
from mijiaAPI_V2.repositories.property_translations import TranslationManager

# 从文件加载
translator = TranslationManager(custom_file=Path("my_translations.json"))

# 使用翻译
cn_name = translator.get_property_translation("my-property")
```

### 4. 动态添加翻译

```python
from mijiaAPI_V2.repositories.property_translations import TranslationManager

translator = TranslationManager()

# 动态添加属性翻译
translator.add_property_translation("new-property", "新属性")

# 动态添加操作翻译
translator.add_action_translation("new-action", "新操作")

# 使用翻译
cn_name = translator.get_property_translation("new-property")  # 返回: "新属性"
```

### 5. 组合使用（推荐）

```python
from pathlib import Path
from mijiaAPI_V2.repositories.property_translations import TranslationManager

# 同时使用外部文件和自定义字典
custom_dict = {
    "properties": {
        "temp-property": "临时属性",
    }
}

translator = TranslationManager(
    custom_translations=custom_dict,
    custom_file=Path("my_translations.json")
)

# 优先级：custom_dict > custom_file > 内置翻译
```

### 6. 导出翻译到文件

```python
from pathlib import Path
from mijiaAPI_V2.repositories.property_translations import TranslationManager

translator = TranslationManager()

# 添加一些自定义翻译
translator.add_property_translation("prop1", "属性1")
translator.add_action_translation("action1", "操作1")

# 导出到文件
translator.export_to_file(Path("exported_translations.json"))
```

## 翻译文件格式

JSON 格式，包含四个类别：

```json
{
  "properties": {
    "English Property Name": "中文属性名"
  },
  "actions": {
    "English Action Name": "中文操作名"
  },
  "types": {
    "bool": "布尔值",
    "int": "整数",
    "uint": "无符号整数",
    "float": "浮点数",
    "string": "字符串"
  },
  "access": {
    "read": "只读",
    "write": "只写",
    "read_write": "可读写"
  }
}
```

## 内置翻译

内置翻译文件位于 `mijiaAPI_V2/repositories/translations.json`，包含常见的属性和操作翻译：

- 设备信息（制造商、型号、序列号等）
- 开关控制（开关状态、电源等）
- 灯光控制（亮度、色温、颜色等）
- 传感器（温度、湿度、PM2.5等）
- 其他常见功能

## 扩展翻译

如果发现某个属性或操作没有翻译，可以：

1. 直接编辑 `translations.json` 文件添加翻译（推荐用于通用翻译）
2. 创建自己的翻译文件（推荐用于项目特定翻译）
3. 通过代码动态添加（推荐用于临时翻译）
